# cellculture-ui

## Documentation cellculture-ui
Open[http://hydra-storybook.local/?path=/story/example-introduction--page](http://hydra-storybook.local/?path=/story/example-introduction--page) to view it in the browser

## How to add to your project

- Make sure your npm already point to https://verdaccio.formulatrix.com, if haven't you can execute below command

```
npm set registry https://verdaccio.formulatrix.com
```

- Recomendation Compitable dependency(CRA/ Create React App) version

```
react-boostrap@0.33.1
react@17.0.1
react-dom@17.0.1
react-script@4.0.1
react-redux@7.2.0
```
